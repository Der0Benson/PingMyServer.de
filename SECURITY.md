# Security Policy

## Supported Versions

Die folgenden Versionen dieses Projekts erhalten Sicherheitsupdates:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

> Bitte verwende immer die neueste Version, um Sicherheitsupdates zu erhalten.

## Reporting a Vulnerability

Wenn du eine Sicherheitslücke in diesem Projekt gefunden hast, beachte bitte:

1. **Keine öffentliche Meldung**
   Bitte erstelle kein öffentliches Issue oder Pull Request mit Details zur Schwachstelle.

2. **Vertrauliche Meldung per E-Mail**
   Sende alle Details an:  
   info@pingmyserver.de

   Bitte gib dabei möglichst an:
   - Betroffene Version
   - Schritte zur Reproduktion
   - Mögliche Auswirkungen
   - Falls vorhanden: Proof of Concept

3. **Reaktionszeit**
   Du erhältst innerhalb von **7 Tagen** eine Rückmeldung.

4. **Weiteres Vorgehen**
   - Nach Bestätigung wird ein Fix vorbereitet.
   - Der Fix wird mit einem Update veröffentlicht.
   - Falls erforderlich, wird das Problem in den Release Notes erwähnt.

Vielen Dank für deine verantwortungsvolle Meldung.
